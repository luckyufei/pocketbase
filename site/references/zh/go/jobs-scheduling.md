# 任务调度

PocketBase 内置支持使用 cron 表达式调度周期性任务。

## 注册 cron 任务

```go
app.Cron().Add("myJob", "*/5 * * * *", func() {
    // 每 5 分钟运行一次
    app.Logger().Info("Running scheduled job")
    
    // 执行某些操作...
})
```

## Cron 表达式格式

cron 表达式格式遵循标准：

```
┌───────────── 分钟 (0 - 59)
│ ┌───────────── 小时 (0 - 23)
│ │ ┌───────────── 月份中的日期 (1 - 31)
│ │ │ ┌───────────── 月份 (1 - 12)
│ │ │ │ ┌───────────── 星期几 (0 - 6)（周日到周六）
│ │ │ │ │
* * * * *
```

示例：
- `*/5 * * * *` - 每 5 分钟
- `0 * * * *` - 每小时
- `0 0 * * *` - 每天午夜
- `0 0 * * 0` - 每周日午夜
- `0 0 1 * *` - 每月第一天午夜

## 删除 cron 任务

```go
app.Cron().Remove("myJob")
```

## 列出 cron 任务

你可以通过 API 列出所有已注册的 cron 任务（仅限超级用户）：

```
GET /api/crons
```

## 手动运行任务

你可以通过 API 手动触发 cron 任务（仅限超级用户）：

```
POST /api/crons/{jobId}
```
